# Desafio Prático - Local Turístico

Nessa aplicação você vai trabalhar com:

- Estrutura HTML;
- Fontes;
- Imagens;
- Alterações das cores dos textos;
- Estilizações com CSS;
- Espaçamentos;
- Listas desordenadas;
- Alterações dos pesos dos textos.
